How the dataset is built

Methodology

This site is a curated GitHub Pages directory, not a full event platform or official publication.

1. Source policy

Prefer official organizer pages, ticketing pages, Meetup event pages, Eventbrite listings, university event pages, and established community portals. Social/community posts are discovery aids, not equal-trust replacements.

2. Inclusion rule

Include events happening in Australia during 2025–2026 that are clearly related to AI, ML, data science, generative AI, AI engineering, analytics, or closely adjacent technical communities.

3. Location policy

Use Australian capitals and nearby major hubs when trustworthy public metadata is visible. City grouping is normalized for cleaner browsing.

4. Dedup logic

Potential duplicates are collapsed using normalized title, date, city, and organizer, preferring the most official canonical URL.

5. Confidence

High confidence means visible title, date, city, and source. Medium confidence is retained only where topic fit is reasonable but source details are less precise.

6. Maps

The map view uses normalized city and hub coordinates for browsing. It is a discovery layer, not precise venue geocoding.

Last-mile curation is manual by design so the data stays human-readable and Git-friendly.
